Desenvolvimento

O desenvolvimento desse front end se deu através das ferramentas fornecidas por HTML, CSS e JavaScript, sem o uso de frameworks ou qualquer outra ferramenta.


O HTML foi utilizado como linguagem de marcação e estruturação principal para o site. O CSS foi utilizado para estilizar todas as páginas disponíveis. O JavaScript foi utilizado para verificação de dados e otimização da experiência do usuário.


O site ainda conta com:

  • 4 páginas HTML;
  • 6 Arquivos CSS;
  • 2 Script JavaScript.
